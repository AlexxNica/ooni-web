Details¶

Test Name: Bridge Reachability Test

Current version: 0.1.2

NetTest: HTTP Requests (https://github.com/TheTorProject/ooni-probe/tree/master/ooni/nettests/blocking/bridge_reachability.py)

Test Helper: None

Test Type: Content Blocking

Requires Root: No

Description¶

This test detects which Tor bridges are working from the given network vantage point and which ones are not. For every bridge specified as input a new tor instance is started and it is instructed to connect to Tor using that bridge. It will try to bootstrap to 100% and either succeed or timeout after a default time of 120 seconds.

How to run the test¶

ooniprobe blocking/bridge_reachability.py -f <input_file>

input file a list of TransportType IP:ORPort bridges (ex. obfs3 198.51.100.45:9045) or IP:ORPort (ex. 203.0.113.27:1828) to test reachability for.

Optional test options¶

-t Specify the timeout after which to consider the Tor bootstrapping process to have failed. The default is 120 seconds.
